# dao

* simple dao factory implementation with support for pluggable implementations
* included basic in-memory and a file-based implementations
* versioned dao specs

###ver. 0 operations
* create
* read
* update
* delete

###ver. 1 operations
* create
* read
* update
* delete
* count
* find
* findOne
* remove

###ver. 2 operations
* create
* read
* update
* delete
* count
* find
* findOne
* remove
* save
* upsert
* bulkCreate
* bulkUpdate
* bulkDelete
